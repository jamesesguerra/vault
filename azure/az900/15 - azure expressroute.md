> extend on-prem network into the cloud with a private connection

## features and benefits
### connectivity to microsoft cloud services
- microsoft office 365
- microsoft dynamics 365
- azure compute services (VMs)
- azure cloud services (azure storage / dbs)

### global connectivity
- enable expressroute global reach and connect on-prem sites from anywhere in the world by connecting your expressroute circuits

### dynamic routing
- expressroute uses bgp for dynamic routing
- bgp allows you to exchange routes between your on-prem network and resources running in the cloud 

### built-in redundancy
- uses redundant devices and multiple circuits

## connectivity models
### co-location at a cloud exchange
- if you’re physically co-located at a cloud exchange, can request to cross-connect to the microsoft cloud

### point-to-point ethernet

### any-to-any networks
- you can integrate your WAN with azure by providing connections to your on-prem 

### directly from expressroute sites
- connect directly to microsoft’s network at a peering location called expressroute direct
