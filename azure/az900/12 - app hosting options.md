## azure app service
- hosting option other than VMs & containers
- enables you to build and host web apps, background jobs, mobile back-ends, REST APIs in the programming language of your choice w/o having to worry about managing the infrastructure
- autoscaling and load balancing
- supports windows & linux
- enables automated deployments from any Git repo (Github, Azure Devops)
- HTTP-based service

### types of app services
- **web apps** - java, php, python
- **api apps** - REST API
- **webjobs** - run a program in the same context as an app; often used to run background tasks as part of app logic
- **mobile apps** - backend for ios and android apps; sdk support
