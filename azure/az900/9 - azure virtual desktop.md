- **azure virtual desktop** - allows you to use a cloud-hosted version of windows from any location
- separates OS, data, and apps from local hardware, running them separately on a remote server
- separate compute environment from user devices so that the risk of confidential info being left on a personal device is reduced
- user sessions are isolated in both single and multi-session environments

### multi session win 10 or 11 deployment
- azure virtual desktop lets you use win 10 or 11 enterprise multi-session, the only win OS that enables multiple concurrent users on a single VM
